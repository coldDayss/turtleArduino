# turtleArduino
거북목증후군을 예방하기 위한 아두이노 코드

센서 구성 및 배치
1. 초음파(5V // trig = D2 // echo = D3)
2. 블루투스(5V // TXD = D8 // RXD= D7)
3. 심박수(5V // A0)
4. 자이로(5V // SDA = A4 // SCL = A5)
